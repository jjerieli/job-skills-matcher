How to read this graph

Each circle is a node — a job title, skill/tool, or certification. Each line is an edge representing a real-world relationship (e.g. "Data Analyst uses SQL"). When you hover a node, its neighbors gently shift outward to reveal the connections, while the hovered node is always drawn on top so its label stays readable. The scoring system counts how many of your target roles share an edge with each skill — the more roles a skill bridges, the higher it ranks.
